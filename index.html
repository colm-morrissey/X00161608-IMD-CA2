2nd Year Project

For my second year project. I worked alongside other classmates to create a film review website. The website allowed users to add their own reviews of movies to the websites database. The user would enter criteria into the website such as the film name, review and genre. This review would then be added alongside other reviews submitted by users. Throughout the project, we used MQTT, Javascript and Google Firebase to host the website.



We used MQTT at the beginning of the project to verify that we were able to make a connection between different computers and send information between them.

Firebase is where the database and the server was hosted. The data stored on Firebase would be called to the webpage using a Javascript API. All data that was entered on the page was also stored on Firebase

We used Javascript to create an API that would be able to contact Firebase and display the contents of the database stored there.
